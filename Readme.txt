CONTENIDO DE LA CARPETA

Esta carpeta está compuesta por lo siguiente:

	-	Carpeta "css": Contiene las hojas de estilos.
	-	Carpeta "images": Contiene las imágenes de prueba utilizadas para el captcha.
	-	Carpeta "js": Contiene todos los scripts.
	-	Archivo "login.html": Archivo (Página) de prueba para mostrar el funcionamiento del captcha.

---------------------------------------------------------------------------------------------------------------------------------------------------------------

COMPLEMENTO CAPTCHA

Está compuesto por 2 archivos principales, que son: 

	1-	captcha-component.js: En este archivo se realiza la creación del componente en la vista y se realizan las validaciones sobre la imagen seleccionada.
							  El archivo se encuentra en la carpeta "js".

	2-	captcha-style.css: En este archivo se encuentran los estilos aplicados al captcha.
						   El archivo se encuentra en la carpeta "css".
